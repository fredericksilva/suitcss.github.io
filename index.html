SUIT CSS

CSS for building self-contained, composable, configurable UI components.

component install suitcss/suit
npm install suitcss
Documentation · Downloads · GitHub

Modular and simple

The naming convention helps to scope your component's CSS and limit the specificity of your selectors. 

/** @define MyComponent */

.MyComponent {}
.MyComponent.is-animating {}
.MyComponent--modifier {}

.MyComponent-part {}
.MyComponent-anotherPart {}

Easy theming

Define default values for themeable properties. Change their compiled value by redefining them in a theme pack. 

/** @define MyComponent */

:root {
  var-color-MyComponent: green;
}

.MyComponent {
  color: var(color-MyComponent);
}

/* ... */

:root {
  var-color-MyComponent: blue;
}

Supporting Node.js tool chain

Package management with Component(1) or npm, preprocessing with Rework, vendor prefixes with Autoprefixer, conformance testing with suit-conformance and html-inspector.
